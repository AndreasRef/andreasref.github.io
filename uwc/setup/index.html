Computer setup before day 3

1) Google Chrome

Google Chrome web browser - download and install


2) p5.js

P5.js online editor - please make an account and make sure you remember your username and password


3) ml5js

ml5js simple example - open this example, press "allow" when it asks you for webcam access, add a 10-20 examples for each class by pressing the "A" and "B" buttons, press "train" and press "start guessing".


Having issues running the example?

    Make sure you are running it in Google Chrome
    Allow camera and microphone in Google Chrome
    Webcam access on Mac: Allow camera on MacOSX in System Preferences
    Webcam access on Windows 10: Select Start > Settings > Privacy > Camera. Set "Let apps use my camera" to "On". Make sure Chrome has access to use the camera.
    Older computers with less powerfull graphics cards might not be able to run the ml5js examples correctly. If this is the case try to borrow another computer or sit two and two on day four.
